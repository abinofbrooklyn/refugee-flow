1. Create `src/components/Admin/LoginForm.jsx`:
   - Functional component with `useState` for `secret` (string) and `error` (string)
   - Accepts prop: `onLogin(secret)` callback
   - Renders a form with:
     - `<h2>Admin Login</h2>`
     - `<input type="password" placeholder="Enter admin secret" value={secret} onChange={...} />`
     - `<button type="submit">Login</button>`
     - If error: `<p style={{ color: 'red' }}>{error}</p>`
   - On submit: calls `onLogin(secret)`. Parent handles auth validation.
   - Basic inline styles: centered form, max-width 400px, padding 20px

2. Create `src/components/Admin/CsvUploader.jsx`:
   - Functional component accepting props: `token` (the admin secret for auth header)
   - State: `file` (File|null), `preview` ({rows, count}|null), `target` (string, default 'route_deaths'), `status` (string: 'idle'|'previewing'|'committing'|'done'|'error'), `message` (string)
   - **Upload + Preview section:**
     - `<input type="file" accept=".csv" onChange={e => setFile(e.target.files[0])} />`
     - `<select value={target} onChange={...}>` with options: `route_deaths`, `war_events`, `asy_applications`
     - `<button onClick={handlePreview} disabled={!file || status === 'previewing'}>Preview</button>`
     - `handlePreview()`: creates FormData, appends file, POSTs to `/admin/csv/preview` with `Authorization: Bearer ${token}`, sets preview state from response
   - **Preview table (shown when preview !== null):**
     - `<p>Parsed {preview.count} rows</p>`
     - `<table>` with `<thead>` from `Object.keys(preview.rows[0])` and `<tbody>` mapping first 20 rows (show "...and N more" if > 20)
     - `<button onClick={handleCommit}>Commit to {target}</button>`
     - `<button onClick={() => { setPreview(null); setFile(null); }}>Cancel</button>`
   - **handleCommit():** POSTs to `/admin/csv/commit` with JSON body `{ rows: preview.rows, target }` and auth header. Shows success/error message.
   - **Status display:** Shows `message` text for success ("Inserted N rows") or error

3. Create `src/components/Admin/AdminPage.jsx`:
   - Functional component with `useState` for `token` (string|null) and `authError` (string)
   - **If no token:** renders `<LoginForm onLogin={handleLogin} />`
   - **handleLogin(secret):** Tests auth by POSTing to `/admin/csv/preview` with auth header. If 401, show error. If 400 (missing file) that means auth passed — set token.
   - **If token:** renders `<h1>Admin Panel</h1>` and `<CsvUploader token={token} />`
   - Wrap in a container div with padding and max-width 1000px

4. Add `/admin` route to the React Router in `src/App.jsx`:
   - Import AdminPage: `import AdminPage from './components/Admin/AdminPage';`
   - Add a `<Route path="/admin" component={AdminPage} />` (or `render` prop depending on existing router pattern)
   - Place BEFORE the catch-all route if one exists
   - NOTE: Check existing App.jsx for the exact React Router v5 pattern used (Route with component prop vs render prop)
