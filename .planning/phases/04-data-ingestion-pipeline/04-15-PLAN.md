    runRules(source, row, config, acceptedIds) function:
    - If source==='iom' and row.id is in acceptedIds set, return [] immediately
    - Geo rules (only if config.hasGeo AND lat != null AND lng != null):
      - Outlier: lat < -90 or > 90 or lng < -180 or > 180 => violation {rule: 'outlier-coordinates', expected: 'lat in [-90,90], lng in [-180,180]', found: `lat=${row.lat}, lng=${row.lng}`}
      - Null island: lat === 0 && lng === 0 => violation {rule: 'outlier-coordinates', expected: 'non-null-island coordinates', found: 'lat=0, lng=0 (null island)'}
      - Geo-label mismatch (only if row.route truthy): call applyGeoBoundsCorrections(row.route, row.lat, row.lng). If result !== row.route => violation {rule: 'geo-label-mismatch', expected: corrected, found: row.route, detail: `coordinates (${row.lat}, ${row.lng}) fall in ${corrected} region`}
    - Value anomaly for countField: parseInt the field. If < 0 => violation. If maxCount set and val > maxCount => violation.
    - Value anomaly for fatField (acled): parseInt row.fat. If < 0 or > maxFat => violation.
    - Value anomaly for dmField (iom): parseInt row.dead_and_missing. If < 0 or > maxDm => violation.

    validateRows(source, rows) async function:
    - Get config from SOURCE_CONFIG[source] (default to {hasGeo: false, countField: null})
    - If source === 'iom', load accepted IDs: query db('data_quarantine').where({source: 'iom', status: 'accepted'}).select('raw_data'), parse each raw_data JSON to extract .id, build a Set of string IDs
    - Wrap the accepted-ID loading in try/catch — on failure, use empty Set and log warning
    - Loop rows, call runRules, partition into clean[] and quarantined[] (quarantined items are {row, violations})
    - Return { clean, quarantined }
    - Wrap the ENTIRE function body in try/catch — on any error, console.error and return { clean: rows, quarantined: [] } (graceful fallback per pitfall #2)

    quarantineRows(source, quarantinedItems) async function:
    - Map items to DB rows: { source, raw_data: JSON.stringify(item.row), rule_violated: item.violations.map(v => v.rule).join(', '), violation_detail: JSON.stringify(item.violations), quarantined_at: new Date().toISOString(), status: 'pending' }
    - Insert into db('data_quarantine')

    module.exports = { validateRows, quarantineRows, SOURCE_CONFIG };
