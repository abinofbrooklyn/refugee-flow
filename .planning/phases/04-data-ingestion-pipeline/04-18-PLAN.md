<action>
1. In `RefugeeRoute_map.tsx`, replace the current canvas-based dot drawing with a MapLibre GeoJSON source + cluster layers:

   a. Create a GeoJSON FeatureCollection from route_death data:
   ```typescript
   const geojson: GeoJSON.FeatureCollection = {
     type: 'FeatureCollection',
     features: routeDeaths.map(d => ({
       type: 'Feature',
       geometry: { type: 'Point', coordinates: [d.lng, d.lat] },
       properties: {
         id: d.id,
         dead_and_missing: d.dead_and_missing,
         cause_of_death: d.cause_of_death,
         date: d.date,
         route: d.route
       }
     }))
   };
   ```

   b. Add the source with clustering:
   ```typescript
   map.addSource('route-deaths', {
     type: 'geojson',
     data: geojson,
     cluster: true,
     clusterMaxZoom: 14,
     clusterRadius: 50
   });
   ```

   c. Add three layers:
   - `clusters` — circle layer for clustered points (filtered by `['has', 'point_count']`)
   - `cluster-count` — symbol layer showing the count number on clusters
   - `unclustered-point` — circle layer for individual points (filtered by `['!', ['has', 'point_count']]`)

   d. Style clusters by count:
   - Small (< 10): 15px radius, color from route color_map
   - Medium (10-50): 20px radius
   - Large (50+): 25px radius

   e. Style individual points using the existing cause-of-death color scheme (preserve the legend colors)

   f. Add click handler on clusters to zoom in: `map.getSource('route-deaths').getClusterExpansionZoom(clusterId)`

   g. Add click handler on individual points to trigger the existing popup/detail view
