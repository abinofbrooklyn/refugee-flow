Create `server/ingestion/acledIngestion.js` with:

1. **getAcledToken()** â€” POST to `https://acleddata.com/oauth/token` with body:
   ```json
   { "email": process.env.ACLED_EMAIL, "password": process.env.ACLED_PASSWORD, "grant_type": "password", "client_id": "acled" }
   ```
   Returns `data.access_token`. Throws `Error('ACLED auth failed: ' + JSON.stringify(data))` if no token.

2. **monthToQuarter(month)** â€” Maps month number (1-12) to quarter string:
   - 1-3 -> 'q1', 4-6 -> 'q2', 7-9 -> 'q3', 10-12 -> 'q4'
   - `return 'q' + Math.ceil(month / 3);`

3. **fetchAcledEvents(token, lastSync)** â€” GET `https://acleddata.com/api/acled/read` with params:
   - `_format=json`
   - `fields=event_id_cnty|event_date|fatalities|latitude|longitude|notes|year`
   - `limit=5000`
   - `page=N` (loop until response data length < limit)
   - If `lastSync` is a Date: add `event_date=${formatDate(lastSync)}|${formatDate(new Date())}&event_date_where=BETWEEN`
   - Format dates as YYYY-MM-DD
   - Authorization header: `Bearer ${token}`
   - Returns flat array of all event objects across pages

4. **transformAcledEvents(events)** â€” Map each event to two output objects:

   **war_events row:**
   ```javascript
   {
     event_id: String(e.event_id_cnty),
     year: String(e.year),
     quarter: monthToQuarter(new Date(e.event_date).getMonth() + 1),
     fat: parseInt(e.fatalities, 10) || 0,
     int: null,
     evt: null,
     cot: null,
     lat: reduceGeoPercision(parseFloat(e.latitude), 2),
     lng: reduceGeoPercision(parseFloat(e.longitude), 2),
   }
   ```

   **war_notes row:**
   ```javascript
   {
     id: String(e.event_id_cnty),
     notes: e.notes || null,
     source: 'ACLED',
   }
   ```

5. **runAcledIngestion()** â€” Main entry point:
   - `const startedAt = new Date();`
   - Get token via getAcledToken()
   - Get lastSync via `getLastSyncDate('acled')`
   - Fetch events via fetchAcledEvents(token, lastSync)
   - Transform via transformAcledEvents(events)
   - Filter out rows where lat or lng is NaN
   - Upsert war_events: `db('war_events').insert(warRows).onConflict('event_id').ignore()` in batches of 500
   - Upsert war_notes: `db('war_notes').insert(noteRows).onConflict('id').merge()` in batches of 500 (merge to update notes if event already exists)
   - Log success: `logIngestion({ source: 'acled', status: 'success', rowsAffected: warRows.length, startedAt })`
   - On catch: `logIngestion({ source: 'acled', status: 'error', errorMessage: err.message, startedAt })`, then re-throw

   Export: `module.exports = { runAcledIngestion, getAcledToken, fetchAcledEvents, transformAcledEvents, monthToQuarter };`

Create `tests/server/ingestion-acled.test.js` with mocked `global.fetch` and mocked `db` (jest.mock for database/connection). Test all behaviors listed above.
