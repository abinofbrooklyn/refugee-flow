Create `server/ingestion/iomIngestion.js` with:

1. **parseCoordinates(coordStr)** — Split "lat, lng" string:
   ```javascript
   function parseCoordinates(coordStr) {
     if (!coordStr || String(coordStr).trim() === '') return { lat: null, lng: null };
     const parts = String(coordStr).split(',');
     if (parts.length !== 2) return { lat: null, lng: null };
     const lat = parseFloat(parts[0].trim());
     const lng = parseFloat(parts[1].trim());
     if (isNaN(lat) || isNaN(lng)) return { lat: null, lng: null };
     return {
       lat: reduceGeoPercision(lat, 2),
       lng: reduceGeoPercision(lng, 2),
     };
   }
   ```

2. **monthToQuarter(month)** — Same as ACLED: `return 'q' + Math.ceil(month / 3);`

3. **transformIomRows(csvRows)** — Map each parsed CSV object:
   ```javascript
   {
     id: String(row['Main ID']),
     date: row['Incident Date'] || null,
     quarter: row['Incident Date'] ? monthToQuarter(new Date(row['Incident Date']).getMonth() + 1) : null,
     year: String(row['Incident Year'] || ''),
     dead: row['Number of Dead'] || null,                    // stored as TEXT
     missing: row['Minimum Estimated Number of Missing'] || null,  // TEXT
     dead_and_missing: row['Total Number of Dead and Missing'] || null,  // TEXT
     cause_of_death: row['Cause of Death'] || null,
     cause_of_death_display_text: row['Cause of Death'] || null,
     location: row['Country of Incident'] || null,
     description: null,
     source: row['Information Source'] || null,
     lat: coords.lat,   // from parseCoordinates(row['Coordinates'])
     lng: coords.lng,
     route: row['Migration Route'] || null,
     route_display_text: row['Migration Route'] || null,
     source_url: row['URL'] || null,
   }
   ```

4. **fetchAndParseIomCsv()** — Download and parse:
   - Fetch `https://missingmigrants.iom.int/sites/g/files/tmzbdl601/files/report-migrant-incident/Missing_Migrants_Global_Figures_allData.csv`
   - Parse response text with `csv-parse/sync`: `parse(csvText, { columns: true, skip_empty_lines: true, trim: true })`
   - Return parsed rows array

5. **runIomIngestion()** — Main entry:
   - `const startedAt = new Date();`
   - Fetch + parse via fetchAndParseIomCsv()
   - Transform via transformIomRows(csvRows)
   - Filter out rows with empty id
   - Upsert: `db('route_deaths').insert(batch).onConflict('id').ignore()` in batches of 500
   - IOM has no date-filter API — always downloads full CSV. `onConflict('id').ignore()` skips existing records efficiently.
   - Log success/error via logIngestion
