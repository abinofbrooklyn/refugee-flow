**server/server.ts — Replace security middleware section (lines 17-21) with Helmet v8 unified config:**

Replace:
```typescript
app.use(helmet());
app.use(helmet.referrerPolicy({ policy: 'no-referrer' }));
app.use(cors());
```

With:
```typescript
app.use(helmet({
  contentSecurityPolicy: {
    useDefaults: false,
    directives: {
      defaultSrc: ["'self'"],
      scriptSrc: ["'self'"],
      styleSrc: ["'self'", 'https:', "'unsafe-inline'"],
      imgSrc: ["'self'", 'data:', 'blob:', 'https://*.basemaps.cartocdn.com', 'https://*.cartocdn.com'],
      fontSrc: ["'self'", 'https://fonts.gstatic.com'],
      connectSrc: ["'self'", 'https://*.basemaps.cartocdn.com', 'https://*.cartocdn.com'],
      workerSrc: ["'self'", 'blob:'],
      childSrc: ["'self'", 'blob:'],
      objectSrc: ["'none'"],
      baseUri: ["'self'"],
      formAction: ["'self'"],
      frameAncestors: ["'none'"],
      upgradeInsecureRequests: [],
    },
  },
  referrerPolicy: { policy: 'no-referrer' },
  crossOriginEmbedderPolicy: false,
}));

app.use(cors({
  origin: '*',
  methods: ['GET', 'HEAD'],
}));
```

Key points:
- `useDefaults: false` — explicit directives for predictability
- `workerSrc: blob:` + `childSrc: blob:` — required for MapLibre GL Web Workers
- `imgSrc` + `connectSrc` include `*.basemaps.cartocdn.com` and `*.cartocdn.com` for map tiles
- `fontSrc: fonts.gstatic.com` — precautionary for Google Fonts (harmless if unused)
- `crossOriginEmbedderPolicy: false` — required for map tile cross-origin loading
- CORS restricts to GET and HEAD only — POST/PUT/DELETE blocked by browser preflight for cross-origin
