For each of the 10 components:

**Conversion pattern (apply to every file):**
1. Read the current .jsx file
2. Delete the .jsx file
3. Create the .tsx file with these transformations:
   - `class X extends React.Component` → `const X: React.FC<XProps> = (props) => {`
   - Define `interface XProps { ... }` with all props the component receives (read from propTypes, constructor, or JSX usage)
   - `this.state = { ... }` → `const [stateVar, setStateVar] = useState<Type>(initialValue);`
   - `this.setState({ key: value })` → `setKey(value)`
   - `this.props.X` → `props.X` or destructure in function signature
   - `componentDidMount()` → `useEffect(() => { ... }, []);`
   - `componentDidUpdate(prevProps)` → `useEffect(() => { ... }, [deps]);`
   - `componentWillUnmount()` → return cleanup function from useEffect
   - `UNSAFE_componentWillReceiveProps(nextProps)` → `useEffect(() => { ... }, [relevantProps]);`
   - `this.methodName = this.methodName.bind(this)` → remove (use const arrow functions)
   - `this.methodName()` → `methodName()` (plain function calls)
   - Remove `import PropTypes` and the propTypes block at bottom (types replace propTypes)
   - If component uses `connect()` + `mapStateToProps`:
     - Remove `connect()` wrapper and `mapStateToProps`/`mapDispatchToProps`
     - Import `{ useAppSelector, useAppDispatch }` from `../../types/redux`
     - Use `const value = useAppSelector(state => state.conflictReducer.propertyName);`
   - If component uses `withRouter6`:
     - Remove `withRouter6` wrapper
     - Import `{ useNavigate, useParams, useLocation }` from `react-router-dom` directly
     - Use hooks instead of props: `const navigate = useNavigate();`
   - Add proper TypeScript imports: `import React, { useState, useEffect, useRef, useCallback } from 'react';`

**Specific notes per component:**
- **DesktopLanding/MobileLanding**: Have intervals in componentDidMount — convert to useEffect with cleanup returning clearInterval
- **Navbar**: May use Redux for route state — convert connect() to useAppSelector
- **Accordion**: Has state for open/closed — convert to useState<boolean>
- **Annotation**: Has styled-components — keep styled-components as-is, just type the component that uses them

**Preserve all visual output exactly** — same styled-components, same CSS classes, same inline styles, same conditional rendering logic. The UI-SPEC mandates zero visual changes.
