**Create `infrastructure/lambda/ip-updater.py`:**

Python 3.12 Lambda function that:
1. Receives EventBridge ECS Task State Change event (lastStatus: RUNNING)
2. Extracts taskArn from event.detail.taskArn
3. Calls ecs.describe_tasks to get ENI attachment
4. Calls ec2.describe_network_interfaces to get PublicIp from ENI Association
5. Calls route53.change_resource_record_sets to UPSERT an A record with TTL=60
6. Environment variables: ECS_CLUSTER_ARN, HOSTED_ZONE_ID, ORIGIN_RECORD_NAME
7. Include error handling with print statements for CloudWatch Logs debugging
8. Add guard: skip if desiredStatus != RUNNING (prevents firing on task shutdown)

Use the exact Lambda code pattern from 14-RESEARCH.md Pattern 3, adding:
- Error handling (try/except with print for CloudWatch)
- Guard check for desiredStatus == RUNNING
